Don't Cross the Line is a level-based puzzle game. Every level has crossed lines which should be uncrossed by player in order to progress.

Try the game

  • 27 handcrafted levels
  • Smooth difficulty curve from very easy levels to really hard
  • Designed for mobile devices
  • Average complete time is 20-30 minutes
  • Tested on iPad, iPad mini, iPhone 4, Samsung Galaxy Note 2
