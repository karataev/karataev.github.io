Description

Your brain is like a muscle, you need to train it to be smart. This little puzzle game is a good brain-builder!

Game's rules are simple:

  1. Connect all the dots in accordance with a level's pattern
  2. You can't go the same line twice
  3. Drawing line should be continious, i.e. no interruptions are allowed.

Features

  • 32 levels with smooth difficulty curve
  • Cute doodle graphics
  • Works perfectly on mobile devices and desktop
